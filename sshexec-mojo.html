wagon:sshexec

Full name:

org.codehaus.mojo:wagon-maven-plugin:2.0.2:sshexec

Description:

Executes a list of commands against a given server.

Attributes:

  • Requires a Maven project to be executed.

Required Parameters

Name Type Since Description
<commands> String[] - The commands that we will execute.
<url> String - URL to upload to or download from or list. Must exist and point to a directory.
User property is: wagon.url.

Optional Parameters

Name Type Since Description
<displayCommandOutputs> boolean - Option to display remote command's outputs.
Default value is: false.
<failOnError> boolean - Allow option not to fail the build on error.
Default value is: true.
<serverId> String - settings.xml's server id for the URL. This is used when wagon needs extra authentication information.
Default value is: serverId.
User property is: wagon.serverId.
<skip> boolean 2.0.0 When true, skip the execution.
User property is: wagon.skip.
